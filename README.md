# bleTester 
仿蓝牙读写器(ble reader)写的一个Android app
可以：
* 搜索设备
* 连接设备
* 发现服务，特征
* 读，写，通知
* (bluetooth low energy reader and writer ,it can search and connect ble devices，find services and characteristics ,can also read and write  characterisrics.)
